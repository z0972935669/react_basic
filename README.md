JSX
1、全稱 javascript XML
2、react定義的一種類似於XML的JS擴展語法: JS+XML
3、本質是React.createElement(component,props,...children)方法的語法糖
4、作用:用來簡化創建虛擬DOM
    a.寫法: var ele = <h1>Hello JSX!</h1>
    b.注意1:它不是字符串，也不是HTML/XML標簽
    c.注意2:它最終產生的就是一個JS對象
5、標簽名任意:HTML標簽或其它標籤
